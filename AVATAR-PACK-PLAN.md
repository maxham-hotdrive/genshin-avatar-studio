## 🛠️ 技术实现

### 文件结构
```
lib/
├── image/
│   ├── processor.ts      # 图片处理（resize, crop, circle mask）
│   └── packager.ts       # ZIP 打包和上传
└── replicate/
    └── client.ts         # AI 图片生成

app/
└── api/
    └── generate-avatar-pack/
        └── route.ts      # 头像包生成 API
```

### API 工作流程
```
1. 用户配置角色 → POST /api/generate-avatar-pack
2. AI 生成 1024x1024 高质量原图
3. Sharp 处理：
   - Resize 到 18+ 种尺寸
   - 应用圆形/方形遮罩
   - 优化压缩
4. 生成 2 张壁纸
5. Archiver 打包成 ZIP
6. 上传到 Supabase Storage
7. 返回下载链接
```

### 成本分析（每次生成）
```
AI 生成成本：     $0.10 (1张图片)
Supabase 存储：   $0.001 (ZIP 约 5MB)
Supabase 流量：   $0.01 (用户下载)
────────────────────────────
总成本：          $0.11

售价：            $20.00
毛利：            $19.89 (99.5%)
```
